# salesagent
Automating process across D&amp;B, HubSpot for sales.

Current Objectives:
Cross reference internal POS and HubSpot for existing contact info/existing customer
Obtain customer contact information
Engage customer in targeted content around prospective product and use case
Call to action for a call or meeting with customer
Offer value prop and value driven content
Assess customer needs through questions and address concerns
Set necessary follow ups
Close sale
